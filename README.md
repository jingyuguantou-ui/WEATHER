# 全球城市实时天气 - Cyber Weather 2077

赛博朋克风格的3D地球天气查询应用

## 功能特性

- 🌍 3D交互式地球 - 支持旋转、缩放、点击城市
- 🔍 城市搜索 - 支持中文/英文城市名搜索
- 🌤️ 实时天气 - 显示温度、湿度、风速、AQI
- 🌙 昼夜切换 - 一键切换日/夜模式
- 🔊 语音播报 - 天气信息语音朗读
- 📱 响应式设计 - 支持桌面和移动端
- 🎨 赛博朋克风格 - 未来科技感UI

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 点击"进入系统"按钮跳过欢迎界面
3. 在搜索框输入城市名（如"西安"、"Beijing"）
4. 点击搜索结果或地球上的城市标记查看天气

## 技术栈

- Three.js - 3D地球渲染
- Web Speech API - 语音合成
- OpenWeatherMap API - 天气数据
- 纯HTML/CSS/JavaScript - 无框架依赖
